# Architectural Decision Framework

HOW to evaluate decisions.

## Decision Process

1. **Gather Requirements** → What does the client need?
2. **Map to Priorities** → Requirement > Cost > Debt > Simplicity
3. **Evaluate Options** → Score each against priorities
4. **Document** → State decision, priority won, trade-off
